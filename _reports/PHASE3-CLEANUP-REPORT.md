### Known Issues / Technical Notes

1. **Windows reserved name `nul`:** `os.path.exists()` always returns True for NUL device. Use `os.listdir()` to verify deletion. `os.remove()` with UNC path `\\?\D:\agent-acceptance\nul` works.

2. **Git porcelain filename escaping:** Files with literal `\n` in names cause `git status --short` to split lines incorrectly. Always use `-z` (NUL-separated) mode for parsing.

3. **GBK encoding:** `subprocess.run(text=True)` defaults to GBK on Chinese Windows. Must specify `encoding='utf-8', errors='replace'`.

4. **PowerShell `$_` in bash:** Bash consumes `$_` before PowerShell receives it. Python scripts are the reliable alternative.
