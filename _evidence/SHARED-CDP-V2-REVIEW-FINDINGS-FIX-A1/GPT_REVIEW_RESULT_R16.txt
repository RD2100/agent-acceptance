R16-BLOCKING-02：SHA256 校验命令失败

你提交的验证命令中包含：

Bash
python -c "
import hashlib
for line in open('hashes.sha256'):
    expected, path = line.strip().split('  ', 1)
    actual = hashlib.sha256(open(path,'rb').read()).hexdigest()
    assert actual == expected, f'{path}: hash mismatch'
print('All hashes verified')
"

我实际运行后发现两个 hash mismatch：

MANIFEST.json
hashes.sha256

其余文件基本匹配，但这两个文件不匹配会导致整条验证命令失败。因此，当前不能说 “All hashes verified”。

这通常是因为 MANIFEST.json 和 hashes.sha256 把自身也纳入 hash 后又被修改，导致自引用 hash 不稳定。解决方式是：从 hashes.sha256 中排除 MANIFEST.json 和 hashes.sha256，或使用 detached manifest checksum。
